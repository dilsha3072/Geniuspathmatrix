# **App Name**: CareerCompass

## Core Features:

- InsightX Assessment Engine: Conduct time-bound assessments for personality, interest, cognitive abilities and self-reported skills.
- AI Career Matching: Uses AI tool to match student profiles to careers via a filtered decision matrix. This feature also provides the option to see SWOT analysis for each matched careers.
- GoalMint Planning Engine: Create custom 1-, 3-, and 5-year plans with academic, skill, and networking goals.
- Progress Tracking: Tracking of academic progress, providing visualizations for users.
- Mentor Booking: Allow students to book time with human mentors. Display the schedule.

## Style Guidelines:

- Primary color: HSL(210, 65%, 50%) which translates to a bright blue (#3399FF), evokes trust and intelligence.
- Background color: HSL(210, 20%, 95%) which translates to a very light blue (#F0F8FF) providing a clean and calming backdrop.
- Accent color: HSL(180, 55%, 40%) which translates to a teal (#33BDBD), to draw attention and suggest growth.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text, ensure legibility.
- Use consistent and intuitive icons that are easy to understand on low-bandwidth devices. Follow Google's Material Design guidelines.
- Mobile-first, accessible layout that adapts to different screen sizes. Follows a clear information hierarchy. Touch targets are large and well-spaced.
- Subtle animations to guide the user through the career discovery journey, keeping it lightweight.
